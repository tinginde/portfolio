使用 Google Places API 來獲取商家評論 — 以台中科博館附近餐廳來練習

本篇練習是想要抓取google map評論，主要參考的是google places api的這兩篇:

https://developers.google.com/maps/documentation/places/web-service/search-nearby?hl=zh-tw

https://developers.google.com/maps/documentation/places/web-service/details?hl=zh-tw

任務: 搜尋台中科博館附近1500公尺的牛排店，然後蒐集其星等及評價

1.先找附近商家的place_id 2.用place_id查找google place 的detail資訊 3.每個商家都有好幾筆評論，用迴圈撈 4.最後存成{place_id: [{rating:text},{rating:text}...]的格式

*本來是把所有查詢結果倒成 reviews.txt {}{}{} 也就是一筆一筆回應之間沒有逗號，後續難處理 **在chatGPT提示下加上ensure_ascii=False，才能把中文結果顯示出來

Required parameters 是location(指該地的經緯度)，以及radius(指要搜尋幾公尺範圍內的商家)

import requests

API_KEY = '自己的apikey'
geo_code = '24.15708879303056, 120.66631531019864'

url = f'https://maps.googleapis.com/maps/api/place/nearbysearch/json?location={geo_code}&radius=1500&type=restaurant&keyword=牛排&language=zh-TW&key={API_KEY}'

payload={}
headers = {}

response = requests.request("GET", url, headers=headers, data=payload)

data = response.content


with open('response.json','wb') as outf:
     outf.write(data)

我把回傳的檔案存成json，方便了解一下有什麼內容

place_id在results裡面，用一個list存起來

import json

API_KEY = '自己的apikey'

# json.loads()將json字串，返回python物件
with open('response.json',encoding="utf-8") as ft:
    data = json.loads(ft.read())

print("The numberof the results:", len(data["results"]))
print("infos from results including:" , data["results"][0].keys())
res = data["results"]

# if we want to get reviews, we have to find out place_id in the results
place_id_group=[]
for i in range(len(res)):
    place_id_group.append(res[i]['place_id'])
# looking for the details
parameters='name,rating,reviews'
payload={}
headers = {}

for place_id_each in place_id_group:

# looking for the details
	url = f'https://maps.googleapis.com/maps/api/place/details/json?place_id={place_id_each}&field="reviews"&reviews_no_translations=true&key={API_KEY}'

	response = requests.request("GET", url, headers=headers, data=payload)

	reviews = response.json()
	id_key = reviews["result"]["place_id"]
	review_list=[]
	for user_idx in range(len(reviews["result"]["reviews"])):
		review_list.append({ "rating" : reviews["result"]["reviews"][user_idx]["rating"]
							, "text" : reviews["result"]["reviews"][user_idx]["text"] })
		reviews_all[id_key] = review_list

with open("reviews.json",'w',encoding='utf-8') as ft:
	json.dump(reviews_all, ft, ensure_ascii=False)
