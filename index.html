Predicting Game Ratings and Identifying Factors Impacting User Reviews Using Machine Learning

We collected the STEAM dataset from Kaggle, extracted features, and applied machine learning models to predict if a new game would receive a rating of over 7 out of 10. The results achieved 75% accuracy. Additionally, using the decision tree method, we identified three factors that impact users' reviews: achievement systems, price, and developers.
